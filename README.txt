Este proyecto incluye:
- Un servlet llamado SaludoServlet que recibe un nombre por GET y lo muestra en pantalla.
- Un formulario HTML que envía datos al servlet.
- Una página JSP con contenido dinámico.
- El archivo web.xml configurado para manejar el servlet.

Instrucciones:
1. Importa este proyecto en un servidor como Apache Tomcat desde tu IDE (NetBeans, Eclipse o VS Code con soporte para Java Web).
2. Asegúrate de que las rutas estén dentro del contexto del servidor.
3. Abre el formulario en: http://localhost:8080/NOMBRE_DEL_PROYECTO/formulario.html
4. Al enviar el nombre, se activará el servlet y mostrará el saludo.
5. Visita la página JSP en: http://localhost:8080/NOMBRE_DEL_PROYECTO/pagina.jsp
6. Incluye este archivo en el comprimido final.
